# Save the Cat (structure pack)

Blake Snyder's 15-beat sheet — the modern default for commercial fiction — as an adoptable [Novel Studio](../../) template: Opening Image through Catalyst, Midpoint, All Is Lost, and Final Image, each with its target position and job.

Requires the Novel Studio app. Install by giving [pack.json](pack.json) to your connected AI; then say "use Save the Cat for \<novel\>" to adopt it (the studio records the adoption in that novel's Blueprint). Scenes tagged with these beat names light up the structure lens in reviews: "which beats have no scene yet?" becomes answerable.

Packs are data, not rules — this installs one `reference` entity (`reference/beat_sheet_save_the_cat`) and touches nothing else.
