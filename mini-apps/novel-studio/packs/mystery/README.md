# Mystery (genre pack)

Fair-play machinery for whodunits, as a [Novel Studio](../../) genre pack. It adds four thread types to the studio's promise/payoff ledger — `clue`, `red-herring`, `alibi`, and `secret`, each with what its payoff owes the reader — and four review lenses: the Fair-Play Check (every clue the solution uses was on the page before the reveal), the Red Herring Audit (every herring gets its innocent explanation), Suspect Coverage (no suspect eliminated offstage), and Reveal Integrity (the solution leans on no detective-only knowledge).

Requires the Novel Studio app. Install by giving [pack.json](pack.json) to your connected AI; then say "adopt the Mystery pack for \<novel\>" (recorded in that novel's Blueprint). From there, tracked clues and herrings become queryable — "which clues has the reader seen so far?" — and reviews run the lenses.

Packs are data, not rules — this installs one `reference` entity (`reference/genre_pack_mystery`) and touches nothing else.
