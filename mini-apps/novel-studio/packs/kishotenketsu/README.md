# Kishōtenketsu (structure pack)

The four-act East Asian structure (起承転結) as an adoptable [Novel Studio](../../) template: Ki introduces, Shō develops, Ten twists, Ketsu reconciles. Its engine is recontextualization rather than conflict — no antagonist required — which makes it the sheet for quiet literary work, slice-of-life, and any story whose ending should feel like the whole settling into place rather than a foe defeated. It also keeps the studio's structure contract honest: a review on this sheet asks "does the Ten reframe what came before?", never "where's the rising conflict?".

Requires the Novel Studio app. Install by giving [pack.json](pack.json) to your connected AI; then say "use Kishōtenketsu for \<novel\>" to adopt it (the studio records the adoption in that novel's Blueprint). Scene beat names use the ASCII spellings Ki, Sho, Ten, Ketsu.

Packs are data, not rules — this installs one `reference` entity (`reference/beat_sheet_kishotenketsu` — the entity name is spelled "Kishotenketsu" so the derived slug stays ASCII) and touches nothing else.
