# Fichtean Curve (structure pack)

Crisis-stacked rising action as an adoptable [Novel Studio](../../) template: open in trouble (no setup act — exposition is smuggled in while things go wrong), climb a staircase of self-contained crises that each leave things worse, crest in a single climax, then land fast. A natural fit for thrillers, horror, and any story that opens mid-fall.

Requires the Novel Studio app. Install by giving [pack.json](pack.json) to your connected AI; then say "use the Fichtean Curve for \<novel\>" to adopt it (the studio records the adoption in that novel's Blueprint). One beat is special: **Rising Crisis** is repeatable by design — many scenes on it is the structure working, and each one's WHAT CHANGES must say how it escalated.

Packs are data, not rules — this installs one `reference` entity (`reference/beat_sheet_fichtean_curve`) and touches nothing else.
